# @dot-event/css

[dot-event](https://github.com/dot-event/dot-event#readme) css

![css](css.gif)

## Install

```js
npm install dot-event @dot-event/css
```

## Setup

```js
const dot = require("dot-event")
require("@dot-event/css")(dot)
```

## Usage

```js
dot.css({ href: "my.css" }) // append css and return <link> element
```
